Oracle Intelligent Bots Advanced Training - Lab 8 (Channels)

Oracle Intelligent Bots provides direct channel support for Facebook, Web, iOS, Android and Webhook as a channel. And other direct channel support is on the roadmap.

In this lab, we'll configure and publish our bot through a web channel, download and install the sample client application, and run and test it against your the bot. Since this allows users to access a through a web page, it’s within easy reach of a multitude of users.

Prerequisites

  1. On the main Bots page in your instance, locate the tile for the advt24horsflowers_bot5, click , select Clone, and name the clone advt24horsflowers_bot5_<YourUniqueInitials>.
  2. Locate the tile for your clone of the bot and click it to open the bot.
  3. Make sure you have Node.js installed on your machine.

Web Channel Setup on Your Local Machine

  1. Open a browser and navigate to the OMCe Downloads page on OTN.

    We'll be using the OMCe version of the testing tools, because the latest AMCe tools download is based on a later version of Node that is not compatible with this lab.

  2. Select the Accept License Agreement radio button.
  3. Download the OMCe Bots Client samples for JavaScript v18.2.3.0 to your machine.
  4. Extract the zip file and navigate to chat-sample-web folder.
  5. Open a terminal window or command prompt in chat-sample-web directory.
  6. Run the following command: 
    npm install
  7. Start the sample ChatSample app by the following command: 
    node server.js

    If your server starts correctly, you should see a message indicating that the server is listening on port 3000.

Configure a New Web Channel for Your Bot

  1. In the left navigation of your bot, click .
  2. Select the Channels tab and click + Channel.
  3. In the Create Channel dialog, give the channel a name and a short description.
  4. Select Web as the Channel Type.
  5. Switch on the Channel Enabled toggle and then click Create.
  6. In the Channel definition screen, copy the App Id to a text file.

    You will use it later on this lab while running the sample client application.
